# EpiEats – Allergy-Friendly Recipe App

EpiEats is a user-centric web platform designed to make cooking safe and inclusive for everyone. It allows users to discover, categorize, and modify recipes based on their specific allergy profiles.

## 🌟 Features

- **Personalized Profiles**: Save your allergy preferences and see them reflected across the app.
- **Smart Filtering**: Recipes are intelligently categorized into Breakfast, Lunch, Dinner, and Desserts.
- **Allergy-Safe Logic**: Ingredient substitution suggestions and clear allergy badges on every recipe.
- **Responsive Design**: A seamless experience across desktop and mobile devices.
- **Secure Access**: Full login and registration system using local storage.

## 🛠️ Built With

- **HTML5 & Vanilla CSS**: For structure and bespoke styling.
- **JavaScript (ES6+)**: Powers the filtering logic and user state management.
- **Tailwind CSS**: For a modern, responsive UI.
- **LocalStorage**: To persist user data and favorites without a complex backend.

## 🚀 Getting Started

Simply open `index.html` in your favorite browser to start exploring recipes!
